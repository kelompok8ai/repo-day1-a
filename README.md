# Corporate Secretary – Public Relations Bank Sumut

Landing page resmi Corporate Secretary Bidang Public Relations Bank Sumut. Dibangun dengan desain modern korporat, responsif, aksesibel, dan siap dikembangkan menjadi website resmi.

## Fitur

- 16 section lengkap sesuai spesifikasi (Hero, Tentang, Visi-Misi, SMART Values, Layanan, Berita, Press Release, Media Center, CSR, GCG, Penghargaan, Statistik, FAQ, Kontak, Footer)
- Sticky navbar dengan blur effect & mega menu
- Dark mode toggle
- Scroll progress indicator & back to top
- Animated counters & scroll reveal
- Press release search & filter
- Masonry media gallery
- FAQ accordion
- Form kontak
- Mobile-first responsive design
- WCAG accessibility (skip link, ARIA labels, keyboard navigation)
- SEO meta tags

## Tech Stack

- HTML5 semantik
- CSS3 (custom properties, glassmorphism, grid, flexbox)
- Vanilla JavaScript (ES modules)
- Vite (dev server & build)

## Menjalankan

```bash
npm install
npm run dev
```

Buka http://localhost:5173

## Build Production

```bash
npm run build
npm run preview
```

Output ada di folder `dist/`.

## Struktur

```
├── index.html          # Halaman utama
├── css/main.css        # Stylesheet lengkap
├── js/main.js          # Interaktivitas & data dinamis
├── public/favicon.svg  # Favicon
├── package.json
└── vite.config.js
```
